#Installation

1. XAMPP instalieren
2. Inhalt von htdocs zum htdocs ordner von XAMPP kopieren
3. XAMPP Control Panel Starten und apache mit mysql starten.
4. zu "http://localhost/phpmyadmin" navigieren und mit user "root" anmelden. Kein PW
5. Zum Tab importieren wechseln und "hueppi.sql" importieren
6. Apache & mysql neu starten.
7. "http://localhost" öffnen.
